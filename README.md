# ☕ Café do Sul

Projeto desenvolvido com o objetivo de aplicar na prática conceitos de **HTML, CSS e JavaScript**, criando uma interface moderna, funcional e responsiva para um cardápio digital de cafeteria.

---

## 📌 Sobre o projeto

O **Café do Sul** é um site que simula o cardápio de uma cafeteria, permitindo ao usuário:

- Visualizar produtos (cafés e assados)
- Adicionar itens ao pedido
- Ver o valor total em tempo real
- Finalizar o pedido

---

## 🛠️ Tecnologias utilizadas

- **HTML5** → Estrutura da aplicação  
- **CSS3** → Estilização e layout responsivo  
- **JavaScript** → Interatividade e manipulação do DOM  

---

## 🎨 Funcionalidades

- 🛒 Adição de produtos ao carrinho  
- 💰 Cálculo automático do valor total  
- 🧾 Exibição dinâmica do pedido  
- 🔄 Reset do pedido ao finalizar  
- 📱 Navegação por âncoras (menu interativo)  
- 🎯 Scroll suave entre seções  

---

## 💡 Aprendizados

Durante o desenvolvimento, foi possível reforçar conhecimentos como:

- Manipulação do DOM com JavaScript  
- Uso de eventos (`addEventListener`)  
- Estruturação semântica com HTML  
- Organização de layout com CSS  
- Lógica de programação aplicada a problemas reais
